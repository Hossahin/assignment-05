Question-1: What are the different ways to select an element in the DOM?

বিভিন্ন ভাবে DOM এর মাধ্যমে আমরা Html element select করতে পারি পারি যেমনঃ getElementsByName, getElementsByClassName, getElementsByTagName, getElementById, querySelector, querySelectorAll ইত্যাদি

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, and textContent এগুলোর মধ্যে মূলত পার্থক্য হলোঃ innerHTML মূলত কোনো element এর ভিতরের Html Tag বা Html Collection কে বুঝায় আর innerText বলতে মূলত কোনো element এর ভিতরের লেখাগুলোকে বুঝাই ।

Question-3: What is event delegation in the DOM?

Event Delegation মানে হলো parent element-এর মাধ্যমে child element গুলোর event handle করা ।

Question-4: What is event bubbling in the DOM?

event bubbling হচ্ছে DOM এ এমন একটা প্রসেস, যেখানে কোনো ইভেন্ট যেমন click, প্রথমে যে element-এ ঘটেছে, সেখান থেকে উপরের দিকে তার parent, এমন করে একে একে উপরের সব parent element এর কাছে ছড়িয়ে পড়ে।

Question-5: How do you create, add, and remove elements using JavaScript?

Javascript এর মাধ্যমে create, add, and remove elements করার জন্য আমরা document.createElement, document.removeChild, append, appendChild ইত্যাদি ব্যবহার করে থাকি ।
